## Описание:
Веб-приложение для профессионального отслеживания и анализа цен на товары.
## Для быстрого запуска:
1. Клонировать/скачать проект
cd priceanalyst/frontend

2. Установить зависимости
npm install

3. Запустить
npm run dev
